## Prerequisites

Before you begin, ensure the following:
- **WooCommerce Plugin Installed**: Your WordPress store must have the WooCommerce plugin installed and activated.
- **Gale Payments Plugin**: You should have received the Gale Payments plugin zip file during your signup process.

---

## Installation Steps

<Steps>
  <Step title=" Upload the Gale Payments Plugin">
   
1. Navigate to your WordPress Admin Panel.
2. Go to `Plugins > Add New > Upload Plugin`.
3. Click `Choose File`, select the Gale Payments plugin zip file, and click `Install Now`.

<img
  className="block "
  src="/images/step1.png"
  alt="Step 1"
/>
  </Step>
  <Step title="Activate the Plugin">

- Once the installation is complete, click `Activate Plugin`.

<img
  className="block "
  src="/images/step2.png"
  alt="Step 2"
/>

  </Step>
  <Step title="Configure Gale Payments Settings">

1. Navigate to `WooCommerce > Settings > Payments`.
2. Locate the **Gale Payments** option and click `Manage`.

<img
  className="block "
  src="/images/manage.png"
  alt="Step 2"
/>


3. Enter your **API Key** provided during signup.
4. Enable **Test Mode** to verify functionality before going live.

<img
  className="block"
  src="/images/step4.png"
  alt="Step 3"
/>
  </Step>

  <Step title="Testing the Integration">
  

1. Add a product to your WooCommerce store.
2. Proceed to checkout and select **Pay with HSA/FSA**.
3. Complete a test transaction using test credentials provided by Gale Payments.
4. Verify the transaction appears in your WooCommerce orders.

<img
  className="block"
  src="/images/step5.png"
  alt="Step 4"
/>
  </Step>


  <Step title="Go Live">


1. Once testing is successful, go back to the Gale Payments settings.
2. Uncheck the **Test Mode** option.

<img
  className="block "
  src="/images/test.png"
  alt="Step 2"
/>

3. **Save your changes.** Your customers can now make payments using their HSA/FSA cards!

  </Step>
</Steps>







## Troubleshooting

- **Plugin Not Visible**: Ensure WooCommerce is installed and updated to the latest version.
- **API Key Issues**: Double-check your API Key for accuracy.
- **Test Mode Transactions Failing**: Ensure you are using valid test credentials provided by Gale Payments.
